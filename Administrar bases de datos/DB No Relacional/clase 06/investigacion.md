# 🎯 Objetivo

Que el alumno:

- comprenda cómo repetir procesos en programación  
- entienda cómo almacenar múltiples datos  
- identifique estructuras clave del lenguaje JavaScript  
- se prepare para trabajar con datos en programas reales  

---

# 📋 Instrucciones

- La investigación debe entregarse en:
  - **hojas de máquina (hojas blancas)**
  - dentro de un **folder**

- El contenido debe estar:
  - **escrito completamente a mano**
  - con sus propias palabras  

- ❌ **No se aceptan trabajos impresos**  
  → Si el trabajo está impreso, **queda automáticamente invalidado**

- Incluir ejemplos cuando se solicite  
- Los diagramas deben estar **dibujados correctamente**  
- No copiar definiciones textuales

# 🧩 Parte 1 — Estructuras de repetición

1. ¿Qué es una **estructura de repetición** en un lenguaje de programación?  
2. ¿Cómo funciona un ciclo `while`?  
3. Realiza un **diagrama de flujo** de un ciclo `while`  

---

# 🧩 Parte 2 — Ciclo `for`

4. ¿Para qué sirve la instrucción `for` en JavaScript?  
5. ¿Qué partes componen un ciclo `for` y qué función cumple cada una?  
6. Realiza un **diagrama de flujo** de un ciclo `for`  
7. ¿Qué diferencia hay entre un ciclo `for` y un `while`?  

---

# 🧩 Parte 3 — Arreglos (arrays)

8. ¿Qué es un **array**?  
9. ¿Para qué sirve un array?  
10. ¿Qué es un **índice** en un array?  
11. ¿Cómo se accede a un dato dentro de un array?  
12. Escribe un ejemplo de un array  
13. Representa gráficamente un array (dibujo)  

---

# 🧩 Parte 4 — Uso conjunto

14. ¿Cómo imprimir un array usando un ciclo `for`?  

---

# 🧩 Parte 5 — Switch / Case

15. ¿Para qué sirve `switch`?  
16. Escribe un ejemplo de `switch/case`  
17. Realiza un **diagrama de flujo** de `switch`  
18. ¿Qué diferencia hay entre `if / else if / else` y `switch`?  

---

# 🧩 Parte 6 — Pensamiento aplicado

19. Explica con tus palabras:

> ¿Cómo usarías un array y un ciclo para buscar un dato?
