# Prácticas — Clase 01
**Tema:** Introducción a JavaScript y Node.js

---

## Práctica 1 — Hola mundo

Crear un archivo llamado `hola.js` con una instrucción que imprima en consola el texto:

    Hola mundo

Ejecutarlo desde terminal con Node.js.

---

## Práctica 2 — Tres mensajes

Crear un programa que imprima tres mensajes diferentes con `console.log()`.

---

## Práctica 3 — Literales de texto y número

Crear un programa que imprima:

- un literal de texto
- un literal numérico

---

## Práctica 4 — Lectura de instrucciones

Tomar la línea:

    console.log("Bienvenidos")

Identificar por escrito:

- objeto
- método
- literal
- instrucción completa

---

## Práctica 5 — Ejecución con ruta completa

Escribir el comando de ejecución usando ruta absoluta del ejecutable de Node.js (simulación), por ejemplo:

    "C:\Program Files\nodejs\node.exe" hola.js

---

## Práctica 6 — Preguntas de repaso

Responder de forma breve:

1. ¿Qué es JavaScript?
2. ¿Qué es Node.js?
3. ¿Qué hace `console.log()`?
4. ¿Qué es un literal?
5. ¿Qué es una instrucción?

---

## Criterios de entrega

- El archivo se ejecuta sin errores.
- Se usa correctamente `console.log()`.
- El alumno identifica los elementos básicos del código.
