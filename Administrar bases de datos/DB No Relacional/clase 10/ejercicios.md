# 📘 Prácticas por equipo — Clase 10 (No Relacional / JS Node.js)

## 🎯 Objetivo

Resolver una práctica por equipo en JavaScript (Node.js), implementando al menos una estructura de control: `for`, `switch/case` o `while`.

## 📌 Lineamientos

- Cada equipo debe entregar **solo su práctica asignada**.
- Deben usar **al menos una** de estas estructuras: `for`, `switch/case` o `while`.
- Pueden usar o no arreglos.
- Pueden leer datos de teclado o usar valores definidos en el código.
- Entrega en carpeta `bdnr/clase-10/` del repositorio.

## Práctica p501 — Menú de conversión de unidades
**Integrantes:** ALCANTAR · SANCHEZ · VELZASCO
**Archivo:** `p501-menu-conversion.js`
**Estructura mínima:** `switch/case`

Crear un menú con opciones:
- 1) Metros a centímetros
- 2) Kilogramos a gramos
- 3) Litros a mililitros

Leer opción y valor, mostrar resultado. Usar `default` para opción no válida.
