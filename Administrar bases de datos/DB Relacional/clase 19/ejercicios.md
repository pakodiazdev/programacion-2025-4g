# Clase 19 — Práctica integradora por equipos

## Mecánica de trabajo

En esta sesión, cada equipo desarrollará una mini-app en PHP CLI con la misma dinámica de la mini-app de tareas:

1. Menú principal con `switch/case`.
2. Arreglo de registros en memoria (array de arrays asociativos).
3. Funciones base para operar el catálogo/directorio/lista.
4. Entrada de datos con `readline()`.
5. Bucle principal hasta elegir salir.

Operaciones mínimas para todas las mini-apps:
- Nuevo registro
- Listar registros
- Eliminar registro
- Salir

No se requiere conexión a base de datos en esta práctica.

## P901 — Catalogo de peliculas (Equipo 1)

Desarrollar una mini-app para administrar un catalogo de peliculas en memoria.

Referencia: [https://github.com/pakodiazdev/tareas](https://github.com/pakodiazdev/tareas). Solo se permite copiar las funciones `centrar` y `titulo`; las demas funciones son solo de referencia.

El sistema debe permitir:
- Registrar una nueva pelicula.
- Listar todas las peliculas registradas.
- Eliminar una pelicula por ID.
- Salir del sistema desde el menu principal.

Campos minimos sugeridos por registro:
- ID
- Titulo
- Genero
- Anio
- Estado (vista/no vista)
