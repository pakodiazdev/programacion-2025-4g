# Ejercicios - Clase 08 (DB Relacional)

## Ejercicios obligatorios

### Ejercicio 1
`clase-08/p01-numero-positivo.php:`

Descripcion:
Programa que reciba un numero y muestre si es positivo.

Si el numero es mayor que 0 mostrar:

Numero positivo

Ejemplo de ejecucion:

```bash
php p01-numero-positivo.php 9
```

Salida esperada:

```text
Numero positivo
```
