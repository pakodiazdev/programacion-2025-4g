# Clase 7 — Ejercicios de laboratorio
**Fecha:** 17 de marzo  
**Materia:** Bases de datos relacionales  
**Tema:** Operadores de comparación, expresiones booleanas e introducción a `if`

---

# Objetivo de las prácticas

Aplicar los conceptos vistos en clase mediante programas simples en PHP ejecutados desde la consola.

A través de estos ejercicios se busca que el alumno:

- practique la lectura de valores desde la línea de comandos
- identifique variables, literales, expresiones y operadores
- utilice operadores de comparación
- comprenda cómo funciona una condición
- use correctamente la estructura `if`
- desarrolle la capacidad de leer, predecir y explicar el funcionamiento de un programa

> En estos ejercicios todavía no se utiliza `else`.
