# Practicas - Clase 09 (DB Relacional)

## Practicas obligatorias

### Practica 31
`clase-09/p31-numero-positivo-o-negativo.php:`

Descripcion:
Programa que reciba un numero y determine si es mayor que 0.

Si el numero es mayor que 0 mostrar:

Numero positivo

En caso contrario mostrar:

Numero cero o negativo

Ejemplo de ejecucion:

```bash
php p31-numero-positivo-o-negativo.php 8
```

Salida esperada:

```text
Numero positivo
```

Ejemplo de ejecucion (rama else):

```bash
php p31-numero-positivo-o-negativo.php -2
```

Salida esperada:

```text
Numero cero o negativo
```
