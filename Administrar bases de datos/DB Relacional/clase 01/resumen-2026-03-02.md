# Conceptos vistos en clase

## ¿Qué es la programación?

La **programación** es el proceso de **crear instrucciones que una computadora puede entender y ejecutar para resolver un problema o realizar una tarea**.

Un programa es un conjunto de instrucciones que indican paso a paso qué debe hacer la computadora.

Estas instrucciones se escriben utilizando **lenguajes de programación**.

Ejemplos de tareas que pueden resolverse con programación:

- realizar cálculos
- procesar información
- automatizar tareas
- crear aplicaciones o sistemas

---

## ¿Qué es un algoritmo?

Un **algoritmo** es una **serie de pasos ordenados y finitos que permiten resolver un problema**.

Los algoritmos no solo existen en programación; también aparecen en actividades de la vida cotidiana.

Ejemplo sencillo de algoritmo:

Preparar una bebida:

1. Tomar un vaso
2. Servir agua
3. Agregar azúcar
4. Mezclar
5. Beber

En programación, antes de escribir código normalmente se piensa primero en **el algoritmo que resolverá el problema**.

---

## ¿Qué es un lenguaje de alto nivel?

Un **lenguaje de alto nivel** es un lenguaje de programación diseñado para que **los humanos puedan leerlo y escribirlo con mayor facilidad**.

Estos lenguajes utilizan palabras, símbolos y estructuras cercanas al lenguaje humano, lo que facilita el desarrollo de programas.

Ejemplos de lenguajes de alto nivel:

- PHP
- Python
- Java
- JavaScript
- C#

Las computadoras realmente solo entienden **lenguaje máquina (binario)**, por lo que los programas escritos en lenguajes de alto nivel deben **traducirse** para poder ejecutarse.
